# Nuxt Modules

Collection of Nuxt modules organized as monorepo for modern application development.

## Modules

| Module | Description |
|---------|-------------|
| analytics | Advanced analytics tracking and reporting |
| command-palette | VSCode-style command palette for quick actions |
| content | Headless CMS with markdown support and SEO |
| email | Email template system and delivery |
| feature-flags | Feature flag management system |
| habbit | Habit tracking and personal development |
| notifications | In-app notification system |
| onboarding | User onboarding and tutorial flows |
| pomodoro | Pomodoro timer and focus management |
| tasks | Task management and project tracking |
| user-acquisition | User acquisition and growth tools |
| waccount | User account management |
| wdocs | Documentation system with search and navigation |
| webhooks | Webhook management and integration |
| wform | Advanced form handling and validation |
| wpayment | Payment processing integration |
| wshop | E-commerce platform |
| wslide | Presentation and slide management |
| wwhiteboard | Collaborative whiteboard |
