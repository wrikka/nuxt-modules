# Nuxt Modules

Collection of Nuxt modules organized as monorepo for modern application development.

## Modules

| Module | Description |
|---------|-------------|
| [analytics](./analytics/) | Advanced analytics tracking and reporting |
| [command-palette](./command-palette/) | VSCode-style command palette for quick actions |
| [content](./content/) | Headless CMS with markdown support and SEO |
| [email](./email/) | Email template system and delivery |
| [feature-flags](./feature-flags/) | Feature flag management system |
| [habbit](./habbit/) | Habit tracking and personal development |
| [notifications](./notifications/) | In-app notification system |
| [onboarding](./onboarding/) | User onboarding and tutorial flows |
| [pomodoro](./pomodoro/) | Pomodoro timer and focus management |
| [tasks](./tasks/) | Task management and project tracking |
| [user-acquisition](./user-acquisition/) | User acquisition and growth tools |
| [waccount](./waccount/) | User account management |
| [wdocs](./wdocs/) | Documentation system with search and navigation |
| [webhooks](./webhooks/) | Webhook management and integration |
| [wform](./wform/) | Advanced form handling and validation |
| [wpayment](./wpayment/) | Payment processing integration |
| [wshop](./wshop/) | E-commerce platform |
| [wslide](./wslide/) | Presentation and slide management |
| [wwhiteboard](./wwhiteboard/) | Collaborative whiteboard |
