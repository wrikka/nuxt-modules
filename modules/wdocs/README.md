# @wrikka/wdocs

Documentation module like VitePress for Nuxt. Built on top of `@wrikka/wcontent` and `@wrikka/wmarkdown`.

## Features

- 📝 Markdown-based documentation
- 🎨 Themeable with WUI components
- ⚡️ Fast static site generation
- 🔍 Full-text search
- 📱 Mobile responsive
- 📊 Mermaid diagrams support
- 🧮 KaTeX math rendering
- 🔗 Link hover previews

## Dependencies

- `@wrikka/wcontent` - Content management
- `@wrikka/wmarkdown` - Markdown parsing & rendering

## Installation

```bash
bun add @wrikka/wdocs
```

## Usage

Add to your `nuxt.config.ts`:

```ts
export default defineNuxtConfig({
  modules: ['@wrikka/wdocs']
})
```
