# Changelog

## 0.0.1

### Initial Release

- Core animation composables: `useAnimation`, `useTimeline`, `useStagger`
- Effect composables: `useFade`, `useSlide`, `useScrollAnimation`
- Animation components: `WAnimation`, `WTransition`, `WStaggerList`, `WReveal`
- Animation utilities: presets, easings, durations, springs
- Anime.js integration with full type support
- Auto-import support for Nuxt 4
