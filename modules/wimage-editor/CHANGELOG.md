# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-03-04

### Added
- Initial release of wimage-editor module
- Canvas-based image editing engine
- Image crop and resize functionality
- Rotate and flip transformations
- Basic filters (grayscale, sepia, blur, brightness, contrast, saturation, hue)
- Undo/Redo history stack
- Touch/Mobile support
- Image upload/download
- Export to PNG, JPEG, WebP formats
- Nuxt 3 module integration
