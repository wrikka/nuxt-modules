# Changelog

All notable changes to this project will be documented in this file.

## [0.0.1] - 2024-03-04

### Added
- Initial release of wvideo-editor
- Video trimming functionality with timeline editor
- Export dialog with format options (MP4, WebM, MOV, GIF)
- Batch export for multiple recordings
- Quality presets for YouTube, Social Media, Web, and Archive
- Recording management interface
- Progress tracking for exports
- Integration with wrecorder module
