## wvideo-editor

Video Editor Module for Nuxt

### Components

- `VideoEditor` - Main video editor interface
- `VideoTrimEditor` - Video trimming with timeline
- `VideoExportDialog` - Export settings
- `VideoBatchExport` - Batch export multiple videos
- `VideoRecordingsList` - Recording management

### Composables

- `useVideoEditor()` - Video editor state
- `useVideoExport()` - Export functionality

### Types

- `Recording` - Video recording data
- `RecordingExportOptions` - Export configuration
- `VideoExportPreset` - Export preset
